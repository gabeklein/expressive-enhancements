# babel-expressive-react-expression-syntax

> Babel preset for full functionality of [Expressive-React](https://github.com/gabeklein/expressive-react) build system.

# Install

```
npm install babel-preset-expressive-react
```

**.babelrc**

```
{
    "presets": [
        "expressive-react"
    ]
}
```
